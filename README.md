# 🛍️ ShopEase - E-Commerce UI

A modern e-commerce frontend built with React.js

1.Features
- 🔍 Search products in real-time
- 🗂️ Filter by category (Phones, Laptops, Shoes, Clothing)
- 🛒 Add to cart with quantity management
- 💰 Dynamic total calculation
- 📱 Responsive design

2. Tech Stack
- React.js
- CSS3
- localStorage

3.Live Demo
[View Live App](https://splendid-meerkat-3487dc.netlify.app)

4. Setup
```bash
npm install
npm start
```
